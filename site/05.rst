###################
SDK 삭제
###################

Setting Tool을 통해서 설치된 Gamebase SDK를 삭제합니다.

Setting Tool 실행
===================

* **Menu > Tools > TOAST > Gamebase > Setting Tool > Settings**

.. image:: _static/image/unity-menu-settingtool.png

삭제
===================

* Click **Remove**

.. image:: _static/image/settingtool-remove.png
    :scale: 50%
